MOTIVATION

Athletes face immense frustration when practising due to time wasted walking around collecting balls.
Our capstone project aims to solve this problem through automation.

Check Out Our Robot!

COLLECTA

Top Speed

The robot is expected to reach a top speed of 2 m/s. That means it can traverse an entire volleyball court in approximately 10 seconds.

Intelligent Ball Detection and Path Planning

State of the art machine learning algorithms are used to detect and plan the optimal path to collect volleyballs.

Lightweight and Portable

Collecta can be moved and used in any indoor volleyball court. Designed to be portable, weighing less than a standard piece of checked baggage.

Quick Ball Retrieval and Storage

The robot collects balls at a fast rate up to 3 volleyballs per run.

DETAILED DESIGN

Robot Chassis
Collecta features a lightweight and modular aluminium extrusion frame.
Detection System
Collecta features a kinect camera for volleyball detection and path planning.
Entry Ramp
Collecta features an entry ramp for smooth and easy volleyball pick-up.
Storage Ramp
The robot has a capacity of 3 volleyballs settled on an incline ramp for easy unloading through a back gate.
Ball Collection System
The robot features a custom volleyball collection system with a cycle time of approximately 2 seconds.
Final Assembly
The final robot assembly for autonomous volleyball detection, collection and retrieval.
